# jsonschema2struct

Tools and libraries to turn **JSON Schema** into **native struct (or type) definitions**—so schemas stay the source of truth and generated code stays easy to read and change.

This repository is under active setup; implementation details and usage will be documented here as they land.

## Development principles

Contributors and AI-assisted edits in this repo follow the **[cognitive load](https://github.com/zakirullin/cognitive-load)** guidelines: optimize for human working memory, prefer clarity over cleverness, and avoid unnecessary indirection. The same ideas are encoded for Cursor in [`.cursor/rules/cognitive-load.mdc`](.cursor/rules/cognitive-load.mdc) (adapted from upstream [`README.prompt.md`](https://github.com/zakirullin/cognitive-load/blob/main/README.prompt.md), CC-BY-4.0).
